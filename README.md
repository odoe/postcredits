# postcredits
Add post credits to your site

## Installation

`npm install postcredits`

## Usage

```js
import { postCredits } from 'postcredits'

const credits = `
    ¯\_(ツ)_/¯

    Developed by: odoe

    Built with: javascript

    Thank you for viewing
`;

postCredits(credits);
```
